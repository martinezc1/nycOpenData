# NYC 311 Service Requests 2010 to 2019

Downloads NYC 311 Service Request from 2010 to 2019 data from NYC Open
Data.

## Usage

``` r
nyc_311_2010_2019(
  limit = 10000,
  filters = list(),
  date = NULL,
  from = NULL,
  to = NULL,
  timeout_sec = 30
)
```

## Source

NYC Open Data:
\<https://data.cityofnewyork.us/Social-Services/311-Service-Requests-from-2010-to-2019/76ig-c548/about_data\>

## Arguments

- limit:

  Number of rows to retrieve (default = 10,000).

- filters:

  Optional list of field-value pairs to filter results (e.g.,
  \`list(agency = "NYPD")\`).

- date:

  Optional single date for \`created_date\` (matches all times on that
  day). Can be a Date or "YYYY-MM-DD" string.

- from:

  Optional start date for \`created_date\` (inclusive). Can be a Date or
  "YYYY-MM-DD" string.

- to:

  Optional end date for \`created_date\` (exclusive). Can be a Date or
  "YYYY-MM-DD" string.

- timeout_sec:

  Request timeout in seconds (default = 30).

## Value

A tibble containing 311 Service Request 2010 to 2019 data.

## Details

This dataset contains all service requests made to NYC 311 from 2010 to
2019. Data is updated daily and includes agency, complaint type,
location, and resolution.

## Examples

``` r
# Examples that hit the live NYC Open Data API are guarded so CRAN checks
# do not fail when the network is unavailable or slow.
if (interactive() && curl::has_internet()) {
  # Quick example (fetch 2 rows)
  small_sample <- nyc_311_2010_2019(limit = 2)
  small_sample

  nyc_311_2010_2019(limit = 2, filters = list(agency = "NYPD", city = "BROOKLYN"))
}
```
